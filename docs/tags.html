HTML Tags Cheat Sheet

Common and important HTML tags with their purpose. Starting with <div> as requested.

Layout & Container Tags

Tag Description
<div> Generic block-level container. Used for grouping content and layout (no semantic meaning).
<span> Generic inline container. Used for styling small pieces of text.
<section> Semantic section of content (thematic grouping).
<article> Independent, self-contained content (e.g., blog post, news article).
<header> Header for a page, section, or article.
<footer> Footer for a page, section, or article.
<nav> Navigation links.
<main> Main content of the document.
<aside> Sidebar or content related to the main content.

Text & Content Tags

Tag Description
<h1> to <h6> Headings (h1 is most important, h6 is least).
<p> Paragraph of text.
<a> Hyperlink (anchor). Use href attribute.
<strong> Strong importance (usually bold).
<em> Emphasis (usually italic).
<br> Line break.
<hr> Horizontal rule (thematic break).
<blockquote> Block quotation.
<pre> Preformatted text (preserves whitespace).
<code> Inline code snippet.

Lists

Tag Description
<ul> Unordered (bulleted) list.
<ol> Ordered (numbered) list.
<li> List item (used inside ul/ol).
<dl> Description list.
<dt> Term in a description list.
<dd> Description/value in a description list.

Tables

Tag Description
<table> Defines a table.
<tr> Table row.
<th> Table header cell.
<td> Table data cell.
<thead> Table header group.
<tbody> Table body group.
<tfoot> Table footer group.
<caption> Table caption.

Forms & Inputs

Tag Description
<form> Form container.
<input> Input field (text, email, password, checkbox, radio, etc.).
<label> Label for a form control.
<button> Clickable button.
<textarea> Multi-line text input.
<select> Dropdown list.
<option> Option inside a select.

Media & Embedded Content

Tag Description
<img> Image. Requires src and alt.
<video> Video player.
<audio> Audio player.
<source> Media source (used inside video/audio).
<iframe> Embed another document (YouTube, maps, etc.).
<svg> Scalable Vector Graphics container.

Document Structure

Tag Description
<!DOCTYPE html> Declares HTML5 document.
<html> Root element of the HTML document.
<head> Metadata container (title, links, scripts).
<body> Visible content of the document.
<title> Page title (in head).
<meta> Metadata (charset, viewport, etc.).
<link> Link to external resources (CSS, icons).
<script> Embed or link JavaScript.

Tip: Always use semantic tags (section, article, header, etc.) when they fit instead of generic <div> for better accessibility and SEO.
